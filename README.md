# Inspection Dashboard

Frontend dashboard for the [Inspection Management API](https://github.com/tiagorcfortunato/inspection-management-api).

Built with vanilla HTML, CSS, and JavaScript — no frameworks or build tools required.

## Live Demo

https://inspection-dashboard.vercel.app

## Features

- User registration and login
- View, create, edit, and delete inspections
- Filter by severity, status, and damage type
- Pagination
- Admin role — see all inspections from all users, including who created each one

## Stack

- **HTML / CSS / JavaScript** — no framework
- **Vercel** — static hosting
- Consumes the [Inspection Management REST API](https://inspection-management-api.onrender.com/docs)

## Running locally

No build step needed. Just open `index.html` in a browser, or serve it with any static server:

```bash
npx serve .
```
